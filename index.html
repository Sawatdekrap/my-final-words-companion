What is this?

This is a companion site for the game "My Final Words". You can use this website to help you play the game.

The key features on this site are (1) a 5-minute timer, (2) an 8-sided die roller, and (3) a link to Solo Mode.

What is My Final Words?

"My Final Words" is a game where you are dying, and you have 5 minutes to write a letter to describe how you died.

This game is still in development, but if you're seeing this you're probably playing an early version of the game.

How do I use the timer?

The 5-minute timer should be started when you start writing your letter. You can play, pause, and reset the timer as you need.

The letter-phase only lasts for 5 minutes, so you'll need to be quick!

How do I use the dice roller?

The dice roller shows the result of an 8-sided die. You can roll the die by clicking on the number.

Roll the die when you need to decide on which symptom applies next. Remember, if the symptom is already taken, use the next-available symptom.

How do I use Solo Mode?

In Solo Mode, the game is played the same way as in the main game until the investigation stage.

In the investigation stage, clicking the Solo Mode link will open a ChatGPT prompt to help you solve the investigation.

Simply take a picture of the letter and ChatGPT will do it's best to solve your murder.

Enjoy!
